About Copin

Copin is a lightweight memo app designed to organize copied text into clean, structured cards. Perfect for AI chat logs, notes, and frequent copy-and-paste workflows.

Features

  • Automatically saves copied text when switching apps
  • Cleans LLM-style spacing and formatting
  • Edit and organize each card individually
  • Sort cards using Boxes for clear categorization
  • Simple, intuitive, distraction-free UI
